# 斗牌Rogue — 游戏玩法机制

## 一、游戏概述

**斗牌Rogue** 是一款以斗地主出牌规则为基础的无尽闯关肉鸽卡牌游戏。

你选择一名角色，每关随机发牌，与镜像AI进行1v1对战。**目标是在每关中以最快的速度清空手牌**。过关后获得新技能，但下一关的敌人将继承你上一关使用的技能——你在和"上一关的自己"战斗。

---

## 二、核心规则：斗地主出牌

### 2.1 牌组

使用54张标准扑克牌（52张 + 大小王），点数从大到小排序：

```
大王 > 小王 > 2 > A > K > Q > J > 10 > 9 > 8 > 7 > 6 > 5 > 4 > 3
```

每关双方各随机发牌，剩余牌进入**抽牌堆**（部分技能可从中抽牌）。

### 2.2 牌型（10种）

| 牌型 | 说明 | 最少张数 | 示例 |
|------|------|----------|------|
| **单牌** | 任意1张 | 1 | 7 |
| **对子** | 2张同点数 | 2 | 7,7 |
| **三条** | 3张同点数 | 3 | 7,7,7 |
| **三带一** | 3张同点数 + 1张单牌 | 4 | 7,7,7 + 3 |
| **三带二** | 3张同点数 + 1个对子 | 5 | 7,7,7 + 3,3 |
| **顺子** | 5张以上连续单牌（3~A，不含2和王） | 5 | 3,4,5,6,7 |
| **连对** | 3组以上连续对子（3~A，不含2和王） | 6 | 3,3,4,4,5,5 |
| **飞机** | 2组以上连续三条，可带翅膀（单牌或对子） | 6 | 3,3,3,4,4,4 |
| **炸弹** | 4张同点数 | 4 | 7,7,7,7 |
| **火箭** | 小王+大王 | 2 | 小王,大王 |

- 飞机带翅膀：可带与三条组数等量的单牌，或等量的对子，或不带翅膀。翅膀数量必须与对手出的飞机一致才能压过。
- 三带一/三带二：跟牌时 kicker 数量必须相同才能压过。

### 2.3 出牌规则

- **轮流出牌**：玩家先手。
- **跟牌**：必须出**同类型**且**更大的牌**才能压过对方。
  - 炸弹可以压任何牌型（包括炸弹之间比点数大小）
  - 火箭可以压任何牌型（包括炸弹），且火箭最大，不可被压
- **过牌（不出）**：无法或不想跟牌时选择"不出"，对方获得自由出牌权（新一轮）。**新一轮时不能过牌**——必须出牌。
- **新一轮**：可出任意合法牌型。
- **胜利**：先清空手牌的一方获胜。

## 三、Run结构

### 3.1 无尽闯关

- 每次Run从**第1关**开始，线性闯关，失败则Run结束。
- 每关玩家发**15张 + 角色加成**张，敌人固定发**15张**。剩余牌进入抽牌堆。
- Run结束后**从第1关重新开始**，无局外成长（纯Roguelike）。

### 3.2 对局流程

```
选角色 → 选初始技能(3选1) → 装备界面 → 第1关战斗 → 胜利 →
  选奖励技能(3选1) → 装备界面 → 第2关战斗 → 胜利 → … → 失败 → 回到主菜单
```

- 初始技能选中后**自动装备到槽位0**。
- 当已获得全部8个技能后，胜利直接进入装备界面（跳过奖励选择）。
- 奖励技能可能出现已拥有的技能（UI会标记"已拥有"），此时仍可选择但无实际收益。

## 四、角色系统

开局前从3个角色中选择1名，每个角色有独特的**固定被动技**：

| 角色 | 被动技 | 效果 |
|------|--------|------|
| **争锋者** | 强袭 | 每回合能量回复+2（普通为+1） |
| **谋略家** | 谋定 | 每关首次切换技能不触发冷却 |
| **掌控者** | 储备 | 手牌上限+2（初始17张，普通为15张） |

---

## 五、技能系统

### 5.1 技能机制

- **获取**：选完角色后和每关胜利后，从随机**3个技能中选1个**获得。
- **装备**：每关开始前在过渡界面装备技能，最多**3个槽位**。
- **释放**：己方回合点击技能槽释放，消耗**能量**。
- **冷却**：使用后进入冷却，每回合开始时CD-1。
- **能量**：初始3点，上限10点。每回合开始时回复（普通+1，争锋者+2）。切换到己方回合时回复。
- **Buff类技能**（S01/S03/S04/S05/S06）：激活后在本回合生效，回合结束自动清除。
- **触发类技能**（S02/S07/S08）：激活后在本回合等待触发条件，触发后即消耗（S02抽牌、S07回能）或限次（S08每回合1次），回合结束未触发则失效。

### 5.2 技能列表

| 编号 | 技能名 | 效果 | 能量 | 冷却 |
|------|--------|------|------|------|
| S01 | 炸弹强化 | 本回合炸弹视为+3级，可压过同级炸弹 | 2 | 2 |
| S02 | 火箭冲刺 | 打出火箭时从抽牌堆额外抽3张 | 1 | 3 |
| S03 | 顺子延长 | 本回合顺子最低长度-1（4张即可出） | 3 | 3 |
| S04 | 连对增幅 | 本回合连对最低组数-1（2组即可出） | 2 | 3 |
| S05 | 三带一强化 | 本回合三带一/三带二可多带一张单牌 | 2 | 1 |
| S06 | 飞机连射 | 本回合飞机最低组数-1（1组即可出） | 3 | 3 |
| S07 | 炸弹馈赠 | 打出炸弹后回复2点能量 | 1 | 2 |
| S08 | 连环炸弹 | 出牌后若手中有炸弹自动打出（每回合限1次） | 4 | 4 |

## 六、敌人：镜像AI

### 6.1 AI行为

- **跟牌时**：搜索所有能压过你的合法出法，优先出点数最小的。炸弹/火箭仅在手牌≤4张时才考虑。
- **新一轮时**：出手中最小的单张牌。
- **延迟**：AI在你出牌/过牌后约0.8秒行动。
- AI难度**不随关卡提升**，始终保持不变。

### 6.2 镜像机制（核心特色）

> **第N关的敌人，使用你在第N-1关实际装备的3个技能。**

- 第1关：敌人**无技能**
- 第2关：敌人使用你在第1关装备的技能（快照在过关时生成）
- 第3关：敌人使用你在第2关装备的技能
- 以此类推…

玩家在过渡界面可以调整下一关的技能装备，但敌人镜像的是**过关那一刻**你装备的技能组合。

敌人释放技能**不消耗能量**，纯冷却制，冷却时间与玩家相同。Buff类技能（S01/S03/S04/S05/S06）对敌人同样生效。触发类技能（S02/S07/S08）敌人在当前版本中实现有限。

### 6.3 策略含义

- 你用强力技能快速过关，下一关就要面对同样的技能组合
- 需要思考技能搭配是否容易被敌人利用
- 适时更换装备的技能是进阶策略

## 七、界面说明

| 界面 | 说明 |
|------|------|
| **主菜单** | "开始游戏"（若已选过角色则直接开始）、"选择角色"、"退出" |
| **角色选择** | 查看3个角色的被动技，悬停有3D透视倾斜动画，点击选择 |
| **关卡过渡** | 左侧：已获得技能列表；右侧：3个装备槽；底部：下一关敌人将继承的技能预览；点击技能再点槽位装备，点击已装备槽位卸下；"开始战斗"按钮进入对局 |
| **战斗界面** | 顶部：敌人手牌（背面）+ 技能槽；中上部：出牌区；中下部：出牌/不出按钮 + 能量条；底部：你的手牌；左下角：3个玩家技能槽 |
| **奖励界面** | 3张技能卡片，悬停上浮+发光，选中1个获得 |
| **失败界面** | 显示到达第几关 + 获得技能数 + 返回主菜单 |

## 八、操作说明

| 操作 | 方式 |
|------|------|
| 选择手牌 | 点击手牌（上浮 + 金色边框 = 已选中，再次点击取消） |
| 出牌 | 选好牌后点击"出牌"按钮 |
| 不出 | 点击"不出"按钮（仅跟牌时可点，新一轮时不可过牌） |
| 释放技能 | 战斗中点击左下角技能槽（CD中或能量不足时无效） |
| 装备技能 | 过渡界面：点击左侧技能再点击右侧空槽 |
| 卸下技能 | 过渡界面：点击已装备的技能槽 |
| 返回主菜单 | 过渡界面/战斗界面左上角"返回"按钮 |
| 重开本关 | 战斗界面按 **R** 键 |
| 全屏切换 | 按 **F11** |
| 快速胜利(开发) | 战斗中点击"我赢"按钮（左上角） |
| 快速失败(开发) | 战斗中点击"我输"按钮（左上角） |

### 8.1 悬停动效

- **手牌悬停**：卡牌平滑上浮 + 选中卡牌金色发光边框
- **按钮悬停**："出牌"/"不出"按钮放大至110%
- **角色选择**：卡牌上浮 + 缩放 + 3D透视倾斜
- **技能奖励**：卡牌上浮 + 缩放 + 发光边框
- 所有动画使用 lerp 平滑过渡

## 九、策略提示

1. **速攻优先**：越早出完手牌越安全，不要囤积大牌。
2. **技能搭配**：炸弹强化(S01) + 炸弹馈赠(S07) + 连环炸弹(S08) 可形成炸弹连发流。
3. **注意镜像**：装备的技能会被下一关敌人继承，谨慎选择强力组合。
4. **角色特性**：争锋者能量充足适合技能型打法；掌控者多2张牌适合持久消耗；谋略家每关可免费切换一次技能，灵活多变。
5. **手牌管理**：S02火箭冲刺抽牌让你更灵活，但也延长了出牌周期；留意抽牌堆剩余数量。
6. **过牌时机**：新一轮时不能过牌，被迫出牌可能打乱手牌结构——必要时宁可拆牌出也不要被动。
