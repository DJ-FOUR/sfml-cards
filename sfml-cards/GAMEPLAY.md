# 斗牌Rogue — 游戏玩法机制

## 一、游戏概述

**斗牌Rogue** 是一款以斗地主出牌规则为基础的无尽闯关肉鸽卡牌游戏。

你选择一名角色，每关随机发牌，与镜像AI进行1v1对战。**你的目标是在每关中以最快的速度清空手牌**。过关后获得新技能，但下一关的敌人将继承你上一关使用的技能来对付你——你在和"上一关的自己做成的自己"战斗。

---

## 二、核心规则：斗地主出牌

### 2.1 牌组
使用54张标准扑克牌（52张 + 大小王），点数排序如下（从大到小）：

```
大王 > 小王 > 2 > A > K > Q > J > 10 > 9 > 8 > 7 > 6 > 5 > 4 > 3
```

### 2.2 牌型（10种）

| 牌型 | 说明 | 最少张数 | 示例 |
|------|------|----------|------|
| **单牌** | 任意1张 | 1 | 7 |
| **对子** | 2张同点数 | 2 | 7,7 |
| **三条** | 3张同点数 | 3 | 7,7,7 |
| **三带一** | 3张同点数 + 1张单牌 | 4 | 7,7,7 + 3 |
| **三带二** | 3张同点数 + 1个对子 | 5 | 7,7,7 + 3,3 |
| **顺子** | 5张以上连续单牌（3~A） | 5 | 3,4,5,6,7 |
| **连对** | 3组以上连续对子（3~A） | 6 | 3,3,4,4,5,5 |
| **飞机** | 2组以上连续三条，可带翅膀 | 6 | 3,3,3,4,4,4 |
| **炸弹** | 4张同点数 | 4 | 7,7,7,7 |
| **火箭** | 小王+大王 | 2 | 小王,大王 |

### 2.3 出牌规则

- **轮流出牌**：玩家先手，轮流进行
- **跟牌**：必须出**同类型**且**更大的牌**才能压过对方
  - 炸弹可以压任何牌型
  - 火箭可以压任何牌型（包括炸弹）
  - 同牌型比主牌点数大小
- **过牌**：无法或不想跟牌时选择"不出"，对方获得自由出牌权（新一轮）
- **新一轮**：可以出任意合法牌型
- **胜利**：先清空手牌的一方获胜

---

## 三、Run结构

### 3.1 无尽闯关

- 每次Run从**第1关**开始，线性闯关
- 每关玩家和敌人各随机发**15张牌**（掌控者角色+2=17张）
- 清空手牌 → 过关 → 进入下一关
- 敌人先清空手牌 → 失败 → Run结束
- Run结束后**从第1关重新开始**，无局外成长（纯Roguelike）

### 3.2 对局流程

```
选角色 → 选初始技能(3选1)→ 装备界面 → 第1关战斗 → 胜利 → 选奖励技能(3选1) → 
  装备界面 → 第2关战斗 → 胜利 → 选奖励技能(3选1) → ... → 失败 → 回到主菜单
```

---

## 四、角色系统

开局前从3个角色中选择1名，每个角色有独特的**固定被动技**：

| 角色 | 被动技 | 效果 |
|------|--------|------|
| **争锋者** | 强袭 | 每回合首次出牌额外+1能量 |
| **谋略家** | 谋定 | 每关首次切换技能不触发冷却 |
| **掌控者** | 储备 | 手牌上限+2（初始22张） |

---

## 五、技能系统

### 5.1 技能机制

- **初始选择**：选择角色后，从随机**3个技能中选1个**作为初始技能（自动装备到槽位0）
- **获取**：击败敌人后，从随机**3个技能中选1个**获得（本Run内永久拥有）
- **装备**：每关开始前，从已获得的技能中选择**最多3个**装备到技能槽
- **释放**：在己方回合点击技能槽释放，消耗**能量**（初始3点，每回合+1），CD中不可用
- **冷却**：有冷却的技能释放后进入CD，每回合开始时CD-1
- **能量上限**：10点

### 5.2 技能列表（8个）

| 编号 | 技能名 | 效果 | 能量 | 冷却 |
|------|--------|------|------|------|
| S01 | 炸弹强化 | 本回合炸弹视为更大，可压过同级炸弹 | 2 | 2 |
| S02 | 火箭冲刺 | 打出火箭时额外抽3张牌 | 1 | 3 |
| S03 | 顺子延长 | 本回合顺子最低长度-1（4张即可出顺子） | 3 | 3 |
| S04 | 连对增幅 | 本回合连对最低组数-1（2组即可出连对） | 2 | 3 |
| S05 | 三带一强化 | 本回合三带一/三带二可额外多带一张单牌 | 2 | 1 |
| S06 | 飞机连射 | 本回合飞机最低组数-1（1组即可出飞机） | 3 | 3 |
| S07 | 炸弹馈赠 | 打出炸弹后回复2点能量 | 1 | 2 |
| S08 | 连环炸弹 | 出牌后若手中有炸弹则自动打出（每回合限1次） | 4 | 4 |

## 六、敌人：镜像AI

### 6.1 基础AI
- 跟牌时优先出最小的合法牌
- 自由出牌时从最小单牌开始出
- AI智能**不随关卡提升**，始终固定

### 6.2 镜像机制
这是本游戏的核心特色：

> **第N关的敌人，将使用你在第N-1关战斗时装备的3个技能**（关卡过渡时快照）

- 第1关：敌人**无技能**
- 第2关：敌人使用你在第1关战斗时的3个技能（快照）
- 第3关：敌人使用你在第2关战斗时的3个技能（快照）
- 以此类推……

玩家在关卡过渡界面可以调整下一关的装备，但敌人镜像的是你**上一关实际使用**的技能组合。
敌人释放技能**不消耗能量**，纯冷却制，冷却时间与玩家相同。

### 6.3 策略含义
- 你用厉害技能快速过关，下一关就要面对同样厉害的技能
- 你需要思考技能搭配是否"容易被敌人利用"
- 适时切换装备的技能是进阶策略

## 七、界面说明

| 界面 | 说明 |
|------|------|
| **主菜单** | "开始游戏"直接选角色进入Run；"选择角色"更换角色 |
| **角色选择** | 查看3个角色被动技，点击选择 |
| **关卡过渡** | 左侧列出的已获得技能，右侧3个装备槽，底部显示下一关敌人将继承的技能 |
| **战斗界面** | 顶部：敌人手牌 + 敌人技能槽；中部：出牌区 + 出牌/不出按钮；底部：你的手牌；左下角：3个玩家技能槽；中下部：能量条 |
| **奖励界面** | 3张技能卡片，选中1个获得 |
| **失败界面** | "你输了" + 到达第N关 + 获得X个技能 + 回到主菜单 |

---

## 八、操作说明

| 操作 | 方式 |
|------|------|
| 选择手牌 | 点击手牌（卡牌上浮 + 金色边框表示选中） |
| 出牌 | 选好牌后点击"出牌"按钮 |
| 不出 | 点击"不出"按钮（仅跟牌时可点） |
| 释放技能 | 战斗中点击底部技能槽 |
| 装备技能 | 关卡过渡界面，点击左侧技能再点击右侧空槽 |
| 卸下技能 | 关卡过渡界面，点击已装备的技能槽 |
| 返回主菜单 | 任何界面左上角"返回"按钮 |
| 重开本关 | 战斗界面按 **R** 键 |
| 全屏切换 | 按 **F11** |
| 快速胜利(开发) | 战斗中点击"我赢"按钮 |
| 快速失败(开发) | 战斗中点击"我输"按钮 |

### 8.1 悬停动效

- **手牌悬停**：鼠标悬停卡牌时，卡牌平滑上浮 + 选中卡牌有金色发光边框
- **按钮悬停**：鼠标悬停"出牌"/"不出"按钮时，按钮放大至110%
- **角色选择**：卡牌悬停上浮 + 缩放 + 3D透视倾斜
- **技能选择**：奖励界面技能卡牌悬停上浮 + 缩放 + 发光边框

---

## 九、策略提示

1. **速攻优先**：越早出完手牌越安全，不要囤积大牌
2. **技能搭配**：炸弹强化(S01) + 炸弹馈赠(S07) + 连环炸弹(S08) 可形成炸弹连发流
3. **注意镜像**：装备的技能会被敌人继承，谨慎选择
4. **角色特性**：争锋者适合技能型打法，掌控者适合持久消耗，谋略家灵活多变
5. **手牌管理**：S02火箭冲刺抽牌可能让你更灵活，但也延长了出牌周期
